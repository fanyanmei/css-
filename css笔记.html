小概念普及

前端三层

html : 结构
css : 样式
JavaScript: 交互 动态效果/数据交互

我们用生活中的一个例子，来解释我们的前端：
html（盖房子）
css（装修房子）
js（空调开关）

网页

网页是网站的基本要素，通俗来讲一个html文件就是一个网页

网站

由很多个网页组成的，共同为一个目标服务的网页集合

站群（流水线统一化管理）

是由很多网站组成的，为同一个公司服务
例如：整个百度旗下子公司有很多网站：百度搜索、百度金融、百度地图、百度音乐….

浏览器

浏览器是指可以显示网页或者显示一个html文件
html、css、js都是运行在浏览器当中
浏览器是一个工厂，html、css、js 是工人和原料
最好用的浏览器是谷歌浏览器

网页

网页的基本组成

html：网页的骨架
css：样式（衣服）
js：动态和交互

html（超文本标记语言）

网页文件本身是一个文本文件（除了文字再无其它），只有通过特定的标记描述，那些普通文本才能在浏览器中正常显示，在页面中遇到图片，音视频的时候如何用文本描述，需要用超文本标记语言进行引入才可以正常显示。

html的兼容问题

因为浏览器是由不同厂商制造的，所以对html 的解析也不一致，产生的页面效果显示也不一致，我们将这个问题叫做兼容性问题。

超文本（标签）

标签的语法

由成对的尖括号和关键字组成，叫做标签
关键字（标签名）：是已经规定过的，不是我们随便制造的。

标签属性

属性提供了有关html元素的更多信息
语法：由属性名和属性值组成的（键值对key=”value”）

一个标签可以有很多标签属性，但是每一个属性之间用空格隔开

<img src="图片" alt="图片描述">

<img> 标签名
<src> 标签属性名
“” 标签属性值

alt的内容只有当文件图片不存在时显示

标签的分类

  • 开始标签 <div>
  • 结束标签 </div>
  • 自闭合标签（空标签） <br/>

html元素

可以将元素理解成一个容器，容器中放置的就是我们网页中的主要内容
是由标签和内容组成的，叫做元素

内容可以为空

<div>内容可以为空</div>

元素的分类

  • 由开始标签和结束标签以及内容组成叫做元素
    • 内容可以为空
      <div>内容</div>
  • 只有一个空元素，只能进行功能上的添加，例如img元素用来引用图片，br用来进行换行<img><br>

标签和元素是用来标记和描述内容的，所以这些标签的元素用户是看不到的，用户只能看到这个元素中的内容

html的基本结构

一个html文件也叫做一个网页，可以称作一个文档（document）
整个文档是从html元素开始，一直向下分支延伸，像一棵大树一样，所以我们将html元素叫做这个文档的“根元素”

<!DOCTYPE html>声明文档

作用：定义这个文档的类型，浏览器先识别这句话，会按照定义的类型去解析这个文档

浏览器是从上到下解析的
所有高版本都会向下兼容，所以在以后的工作中我们直接将文档声明写成 html5 就可以了
声明文档不区别大小写
如果html文件没有写文档声明，会触发浏览器的怪异模式。
文档声明不是html标签
声明文档必须写在 html 页面的第一行

<!DOCTYPE html> 是html5 的声明文档

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"

"http://www.w3.org/TR/html4/loose.dtd">

是html4的声明文档

选择器分类

标签选择器 格式 标签

直接将HTML元素当作选择器来用
权重：1

    <style>

         h1 {color: red}

    </style>

   <h1>我是h1</h1>

类选择器 格式 .标签

将html元素的class标签属性值，当做选择器来用，需要在这个标签属性值前面加 .
权重：10

一个html元素可以有很多个class标签属性值（类名）
每一个类型之后用空格隔开
类名可以重复使用

    <style>

    .box{background-color: #cccccc}

        .box1{background-color: green}

    </style>

    <div class="box box1">我是div</div>

id选择器 格式 #标签

将html元素的id标签属性值，当做选择器来用，需要在这个标签属性值前面加 #
权重：100
使用场景：锚点或者js动态交互

   <style>

    .box.box1{background-color: #cccccc;

               background-color: green}

     #div1{background-color: aqua}

    </style>

    <div id="div1" class="box box1">我是div</div>

通配符选择器 格式 *{属性名：属性值}

匹配全部的html元素
一般不使用，全部匹配比较消耗性能
权重：0<*<1

通配符选择器在工作当中我们不用这个选择器

<style>

     *{属性名：属性值}

    </style>

我们在工作中，常用的是类选择器、标签选择器
id选择器需要用到锚点、js动态效果的时候，在使用
id选择器它具有唯一性，一个页面id属性值只能出现一次
标签选择器相当于人的姓，类选择器相当于人的名字，id选择器相当于人的身份证号

格式

伪元素

通过css代码向指定元素“内”添加假的（html中不存在）的元素

before 会出现在元素所有内容之前
after 会出现在元素所有内容之后

使用伪元素的时候要保证两个前提:

  • 要有display： 这个属性
  • 要有content这个属性，这个属性的属性值可以为空，但是引号不能少–content:”” 
<style>

  body,ul{margin: 0;  padding: 0}

        li{

            color: red;

            font-weight: bold;

            list-style: none;



        }

        li:before{

            display: inline-block;

            content: url("img/gougou.jpg");

            color: green;

        }

        li:after{

            display: inline-block;

            content:"我也是li";

            color: green;

        }

    </style>

定位

相对定位position:relative

特点

  • 发生位置改变时，原来的位置还占用
    • 不脱离文档流
  • 层级大于文档流的其他元素（会盖在其他元素之上）

  • 给绝对定位的元素做参 照物（常用）
  • position：relative
    • 当top和bottom 同时设定时，top生效
    • 当left和right同时设定时，left生效

盒子自己发生位置改变，又不影响其他元素的时候，选择相对定位

绝对定位position：absolute

特点

  • 脱离文档流
  • 在不设置参照物时，参照物是body
  • 所有元素默认都会去找参照物的起点位置，相互叠盖在一起
  • 人为设置参照物必须满足三个条件
    • 需是这个绝对定位元素的父级元素
    • 参照物必须带有定位属性的元素
      • 定位属性（相对、绝对、固定）
    • 同级元素不能作为参照物
  • 父级元素都有定位属性时，绝对定位参照物是离它最近那个的父级元素
  • 绝对定位的元素有宽高的时候，top和bottom的值同时设置的时候，top值生效。left和right的值同时设置，left生效
  • 在不设置宽高时，宽度和高度是由本身内容决定的
  • 当绝对定位元素如果不设置4个方向的值并且同级之前有其他内容（元素），它会默认排到内容（元素）之后
  • 绝对定位设置宽高为100%，继承参照物的宽高

固定定位position:fixed

  • 脱离文档流
  • 参照物是浏览器的可视窗口
  • 在不设置宽高时，宽高也是由内容决定的
  • 多用在页面中的辅助导航、或者返回顶部等位置
